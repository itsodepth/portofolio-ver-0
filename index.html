Halo! Perkenalkan nama saya Muh Devano Alfarizy. Saat membuat portofolio ini, saya masih berstatus mahasiswa semester 2 di Universitas Duta Bangsa yang lokasinya ada di Surakarta, Jawa Tengah. Saya membuat portofolio versi 0.0 ini sebagai bentuk latihan saya membuat portofolio yang bagus dan rapi.

Mungkin kalian bertanya-tanya, apakah saya sudah pernah membuat sebuah website yang telah jadi seutuhnya untuk client sebelumnya? Belum, saya belum pernah membuat sebuah website yang sudah jadi seutuhnya untuk client. Saya adalah seorang pemula yang baru berniat belajar Front-End. Sebelumnya saya sudah pernah belajar Front-End Web Development saat duduk di bangku SMK, namun waktu itu niat saya masih setengah-setengah.
